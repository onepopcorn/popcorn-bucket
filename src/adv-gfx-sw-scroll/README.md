# GFX Software based scroll
An example of software based scroll using a virtual camera.

## What you learn
This example shows how to use a software based scroll approach by using a backbuffer (in RAM) bigger than the visible area (VRAM) by only blitting the parts that should be visible on the screen while udpating edge off-screen colums to create the scroll effect. This approach uses the concept of a "virtual camera" to encapsulate the logic.
